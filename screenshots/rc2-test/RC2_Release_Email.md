Dear Valued User,

We're excited to announce **Release Candidate 2** of the Potomac Analyst Platform! This major update brings a suite of powerful AI-driven tools designed to supercharge your systematic trading workflow.

## What's New in RC2

### 🧠 Enhanced AI Chat Experience
Our AI assistant, **Yang**, now features:
- **Real-time streaming responses** — Watch answers generate in real-time
- **Voice conversation mode** — Speak your queries naturally
- **File upload support** — Drop PDFs, CSVs, AFL files, images, and documents directly into chat
- **Knowledge Base integration** — Reference your uploaded documents for context-aware answers
- **Chain of Thought visualization** — See which tools the AI is using to answer your questions
- **Model selection** — Choose between different AI models for optimal results

### 📊 AFL Code Generator (Major Upgrade)
Transform your trading ideas into production-ready AmiBroker code:
- **Plain language to AFL** — Describe any strategy in English and receive optimized code
- **Composite Mode** — Build multi-strategy systems with automatic signal aggregation and majority voting
- **Built-in Monaco Editor** — Professional code editing with syntax highlighting
- **Backtest Settings Panel** — Configure equity, position sizing, commissions, and trade delays
- **Quick Actions** — Optimize, debug, and explain generated code with one click
- **Strategy type selection** — Standalone, Entry, or Exit strategies

### 🔍 Backtest Analysis (New)
Decode your backtest reports with AI-powered breakdowns:
- Upload backtest HTML reports for instant analysis
- Get actionable insights on performance metrics
- Identify optimization opportunities

### 📚 Knowledge Base (New)
Your documents, intelligently searchable:
- Upload trading documents, PDFs, and archives
- Semantic search powered by AI
- Reference documents directly in chat conversations
- Context-aware responses based on your uploaded materials

### 🎯 Reverse Engineer Tool (New)
Convert strategy logic directly into AFL code:
- Paste strategy descriptions or pseudo-code
- Get working AmiBroker implementations
- Maintain logical structure from source material

### 🎨 Redesigned Dashboard
A completely refreshed command center:
- **Real-time activity stats** — Track conversations and documents at a glance
- **Quick action buttons** — Jump to any tool instantly
- **Feature cards with sparkline data** — Visual performance indicators
- **Platform guide** — Onboarding tips for new features
- **Animated ticker bar** — Live platform status indicators

### ⚙️ Agent Settings (New)
Fine-tune the AI's behavior:
- **Thinking effort control** — Adjust from minimal to maximum reasoning depth
- **Prompt caching** — Faster responses for repeated queries
- **Max iterations** — Control tool execution limits
- **Model version pinning** — Lock to specific AI model versions
